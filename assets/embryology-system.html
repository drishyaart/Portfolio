Interactive Embryology System

Objective

Visualize cellular self-organization during early embryonic development.

Approach

Built a multi-software pipeline using Houdini, 3ds Max, Maya, and Unity to simulate dynamic biological processes.

Key Decisions

Focused on spatial accuracy and temporal progression. Modeled dynamic interactions to represent emergent biological behavior.

Outcome

Developed an interactive system that allows exploration of complex developmental processes.

Tools

Houdini, Unity, 3ds Max, Maya
